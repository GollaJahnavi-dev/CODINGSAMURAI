## 📌 Tasks Completed

### 1️⃣ Library Management System

A Java console-based application that simulates a simple library system.

**Features**

* Add books to the library
* Borrow books
* Return borrowed books
* Display available books
* Track borrowed books by users

**Concepts Used**

* Java Classes & Objects
* ArrayList
* Object-Oriented Programming (OOP)
* Conditional Statements
* Loops

---

### 2️⃣ Online Quiz Application

A Java-based console quiz application that allows users to answer multiple-choice questions and view their results.

**Features**

* Multiple choice questions
* Instant answer validation
* Score calculation
* Quiz summary with correct answers
* Retry option for users

**Concepts Used**

* Classes & Objects
* Lists (ArrayList)
* User Input using Scanner
* Conditional Logic
* Basic Java Program Structure
